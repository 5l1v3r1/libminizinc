MiniZinc 2.1.0beta with MiniSearch
==================================

This package contains the MiniSearch meta-search language and includes the
MiniZinc 2.1 constraint modelling language and tool chain. It provides 
backends for all FlatZinc solvers.

** Documentation **

For a detailed documentation please visit:
http://www.minizinc.org/minisearch

** Bugs **

If you encounter any problems with MiniZinc, please use the MiniZinc bug tracker
at http://www.minizinc.org/trac to report any issues or feature requests.
